Anzan Trainer PWA

Files:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- sw.js
- icons/

How to host:
- Upload the whole folder to any static web host.
- For offline support and installability, host it over HTTPS (or use localhost during local development).

iPhone:
- Open the site in Safari.
- Tap Share > Add to Home Screen.

Notes:
- This app is a static PWA.
- It uses the same problem shapes as your current Python generator, but not Python's exact random sequence.
- Redo-misses mode recycles wrong answers until correct.
